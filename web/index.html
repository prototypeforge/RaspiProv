Connect to your Raspberry Pi — without the hassle.

Pair with a Pi running pi-ble-config over Bluetooth and configure it from this page: pick a Wi-Fi network, toggle SSH, check the device's IP — all without plugging in a cable, sharing a network, or memorising an IP.

No cloud, no account, no router setup. Just click the button.

Open source — grab it at github.com/prototypeforge/RaspiProv.

  • 📶 Scan & join Wi-Fi
    Pick from a live list, sorted by signal.
  • 🔐 Toggle SSH
    Enable remote shell when you need it.
  • 🌐 See every interface
    Ethernet, Wi-Fi, IP and link state — at a glance.
Pi not showing up?
  • Make sure the agent is running on the Pi: sudo systemctl status pi-ble-config.
  • Look for PiCfg-<hostname>-<XXXX> in the browser's BLE chooser.
  • On Linux PCs running Chrome from Snap/Flatpak, Web Bluetooth is blocked — install Chrome from the official .deb instead.
  • If navigator.bluetooth is missing, enable Web Bluetooth in chrome://flags and relaunch.
